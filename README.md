# Valentine Memories

A single-page Valentine experience with a sweet Q&A, a love meter, floating photos, and a final proposal screen.

## Features
- Guided Q&A with score-based love meter.
- Memory gallery with photos and video.
- Playful final prompt with yes/no interaction.
- Background music and floating photo cloud.

## Project structure
- `index.html` — main layout and content.
- `style.css` — visual styling and animations.
- `script.js` — interactivity and love meter logic.
- `assets/` — photos, audio, and video (add your own media here).

## Usage
Open `index.html` in any modern browser. Replace items in `assets/` to personalize the experience.

## Customization tips
- Update the question text in `index.html`.
- Adjust animation and colors in `style.css`.
- Tweak scoring values in `script.js`.
